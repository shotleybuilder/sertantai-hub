# API Documentation

## Health Check

### GET /health

Returns service health status.

**Response:**
```json
{
  "status": "ok",
  "service": "sertantai-hub",
  "timestamp": "2024-01-01T00:00:00Z"
}
```

### GET /health/detailed

Returns detailed health status including database connectivity.
